How it works

Three ways to integrate.
Start with zero effort.

The proxy works with any language and any framework. Use the SDK only when you need more control.

Recommended
Option 1
Proxy — zero code changes

Change your LLM base URL. That's it. AgentLens forwards requests to OpenAI/Anthropic and captures everything automatically.

# Before
OPENAI_BASE_URL=https://api.openai.com

# After
OPENAI_BASE_URL=http://localhost:8090/v1/p/{id}/openai
Option 2
SDK — one import

Auto-patches your OpenAI or Anthropic client. Adds richer metadata and custom span names.

// npm i @farzanhossans/agentlens-openai

import '@farzanhossans/agentlens-openai'

// done. every call is traced.
Option 3
Manual — full control

Wrap any code in AgentLens.trace() for custom spans, metadata, and parent/child hierarchies.

// Full control over spans

AgentLens.trace('classify', (span) => {
  span.setMetadata('user', id)
  // your LLM calls here
})

# Clone and configure
git clone https://github.com/farzanhossan/agentlens
cd agentlens/infra
cp .env.prod.example .env

# Generate secrets
openssl rand -hex 32  # paste as JWT_SECRET
openssl rand -hex 32  # paste as HMAC_SECRET

# Start everything
docker compose -f docker-compose.prod.yml up -d --build

# Dashboard  http://localhost:4021
# API        http://localhost:4020
# Proxy      http://localhost:8090
