Integration

Integrate in 60 seconds.

npm install @agentlens/core @agentlens/openai
import { AgentLens } from '@agentlens/core'

AgentLens.init({
  apiKey: 'proj_xxx',
  projectId: 'my-agent'
})
import '@agentlens/openai'
// done. every OpenAI call is now traced.

// Works with Anthropic too:
import '@agentlens/anthropic'

That's it. No YAML. No config files. No agent rewrites.
